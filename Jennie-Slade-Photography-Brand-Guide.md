# Jennie Slade Photography — Brand Guide

## Design Inspiration

Inspired by **Lauren Fair Photography** and **Tec Petaja Photography**. Editorial, refined, airy. Think high-end fashion magazine, not business website.

---

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| **Warm Taupe** | `#4b4746` | Primary text, buttons |
| **Warm Gray** | `#8a8483` | Secondary text, labels |
| **Warm Gray Medium** | `#d4d1cb` | Dividers, placeholder images |
| **Warm Gray Light** | `#e3e0da` | Alternate section backgrounds |
| **Cream / Ivory** | `#f8f4ef` | Primary background |
| **Warm White** | `#fdfcfa` | Light backgrounds |
| **Teal Sage** | `#5f8f8b` | Accent (use sparingly) |
| **Blue Mist** | `#b8c5d4` | Soft accent (rarely used) |

**Rule:** All grays are WARM (beige/yellow undertone). Never use cool or blue grays. Text is warm taupe, not black.

## Typography

| Role | Font | Weight | Style |
|------|------|--------|-------|
| **Display / Titles** | Cormorant | Light (300) | Uppercase, 0.1em letter spacing |
| **Body Text** | Lora | Regular (400) | Sentence case, 1.9 line-height |
| **Labels / Headings** | Montserrat | Light (300) | Uppercase, 0.15em letter spacing |
| **Accent / Testimonials** | Arapey | Regular (400) | Italic, 18px |
| **Subheadings / Buttons** | Apparel | Regular (400) | Uppercase, 0.12em letter spacing |
| **Script** | Destiny | Regular | Decorative use only |

**Font sources:**
- Cormorant, Lora, Montserrat, Arapey — Google Fonts (free)
- Apparel — Fort Foundry (premium, self-hosted)
- Destiny — Custom script (self-hosted)

**Key principle:** Titles use Cormorant LIGHT (weight 300). The delicacy IS the aesthetic. Never bold.

## Brand Voice

- **First person:** "I'd love to capture your story"
- **Warm, conversational, story-first**
- **Never uses:** em dashes, "leverage," "optimize," "utilize," "synergy," "platform"
- **Short paragraphs:** 2-3 sentences max
- **Rhetorical questions welcome**
- **Encouraging, never pushy**

## Design Principles

1. **Photography is the product.** Design frames photos, never competes.
2. **Warm, not cool.** Warm taupe, cream, warm grays. Never cold.
3. **Airy, not cramped.** When in doubt, add more whitespace.
4. **Light, not heavy.** Cormorant Light, not bold. Delicacy IS the aesthetic.
5. **Refined, not flashy.** Subtle animations, restrained palette, editorial restraint.
