    6. Message — textarea, 5 rows minimum, required (per D-22)

    - Display errors below each field in `text-sm` with warm red tone: use `text-red-700` (not bright red). Error messages per D-05 — warm wording: "Please enter your name", "Please enter a valid email address", "Please select a session type", "Please tell me a bit about what you're looking for", "Please include a message so I know how to help"
