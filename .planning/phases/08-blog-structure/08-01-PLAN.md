2. Create `content/posts/.gitkeep` — empty directory for MDX blog posts.

3. Create `src/lib/blog.ts` with these exports:

```typescript
export interface BlogPostMeta {
  title: string
  date: string        // ISO date string (e.g., "2026-03-15")
  excerpt: string     // 1-2 sentence summary for listing page
  featuredImage: string  // Cloudinary public ID (per BLOG-04)
  slug: string        // Derived from filename
}

export interface BlogPost {
  meta: BlogPostMeta
  content: string     // Raw MDX content (parsed by next-mdx-remote on render)
}
```

Functions:
- `getAllPosts(): Promise<BlogPostMeta[]>` — reads all .mdx files from `content/posts/`, parses frontmatter with gray-matter, returns array sorted by date descending (newest first). Uses `process.cwd()` + `path.join` to resolve the content directory. Returns empty array if directory is empty or has no .mdx files.
- `getPostBySlug(slug: string): Promise<BlogPost | null>` — reads a single .mdx file by slug (filename without .mdx extension), returns parsed frontmatter + raw MDX content. Returns null if file not found.

The slug is derived from the filename: `my-first-post.mdx` becomes slug `my-first-post`.

Use Node.js `fs/promises` and `path` modules. This runs server-side only.
