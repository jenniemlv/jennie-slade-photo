Create `src/components/gallery/GalleryGrid.tsx` as a `'use client'` component. This is the core gallery component used by all three gallery pages. Per D-17, D-21, D-22.

**Component structure:**

```typescript
'use client'

import { useState } from 'react'
import Lightbox from 'yet-another-react-lightbox'
import 'yet-another-react-lightbox/styles.css'
import ScrollFade from '@/components/ui/ScrollFade'
import type { GalleryImage } from '@/data/galleries'

interface GalleryGridProps {
  images: GalleryImage[]
  aspectRatio: '3/2' | '2/3'
}
```

**CRITICAL — Tailwind v4 aspect-ratio lookup (from project convention and Pitfall 1 in RESEARCH.md):**
```typescript
// MUST use Record lookup — never string interpolation for Tailwind classes
const aspectClasses: Record<'3/2' | '2/3', string> = {
  '3/2': 'aspect-[3/2]',
  '2/3': 'aspect-[2/3]',
}
```

**Component body:**

1. State: `const [lightboxIndex, setLightboxIndex] = useState(-1)` — negative means closed per D-13.

2. Slides array for YARL: Map images to `{ src: img.src || '/images/placeholder.svg', alt: img.alt }`. When `src` is empty (placeholder mode), the lightbox won't show meaningful images, but the click handler still works structurally.

3. Grid: `<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4 md:gap-6">` per D-08.

4. Each grid cell:
   - Wrapped in `<ScrollFade key={image.id}>` — individual cells, NOT the whole grid (Pitfall 4 in RESEARCH.md)
   - `<button onClick={() => setLightboxIndex(index)} className="block w-full cursor-pointer group" aria-label={`View ${image.alt} full screen`}>` per Pitfall 5
   - Inside button: `<div className={`relative ${aspectClasses[aspectRatio]} bg-warm-gray overflow-hidden`}>` — CLS prevention per D-11, D-27
   - If `image.src` is not empty, render a Next.js Image inside the aspect-ratio div with `fill`, `className="object-cover group-hover:opacity-90 transition-opacity duration-150"`, `sizes="(max-width: 768px) 100vw, (max-width: 1024px) 50vw, 33vw"`, and `loading="lazy"` per D-28
   - If `image.src` is empty, the warm-gray background is the placeholder per D-26

5. Hover effect (Claude's discretion): Subtle opacity reduction on hover — `group-hover:opacity-90 transition-opacity duration-150`. Not scale, not overlay. Just a slight dim.

6. Lightbox: Render `<Lightbox>` after the grid div:
   ```tsx
   <Lightbox
     open={lightboxIndex >= 0}
     index={lightboxIndex}
     close={() => setLightboxIndex(-1)}
     slides={slides}
   />
   ```
   Keyboard nav (left/right/escape) and swipe are ON by default per D-15. No extra config. No thumbnail strip per D-14. Click outside closes per D-16 (YARL default).

**Do NOT:**
- Use `'use client'` on any page file — only GalleryGrid needs it
- Wrap the entire grid in a single ScrollFade (Pitfall 4)
- Construct aspect-ratio classes dynamically with template literals (Pitfall 1)
- Use `priority` on any gallery image (Pitfall — all lazy per D-28)
- Use `<img>` tags — use Next.js `Image` or `CloudinaryImage` when real images are ready
