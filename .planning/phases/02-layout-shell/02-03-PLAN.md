2. Update `src/app/layout.tsx` to wire in Header, Footer, PageTransition, and skip-to-content link:

   **Add imports:**
   ```tsx
   import Header from '@/components/layout/Header'
   import Footer from '@/components/layout/Footer'
   import PageTransition from '@/components/ui/PageTransition'
   ```

   **Update the body JSX to:**
   ```tsx
   <body>
     {/* Skip-to-content link for keyboard accessibility */}
     <a
       href="#main-content"
       className="sr-only focus:not-sr-only focus:fixed focus:top-4 focus:left-4 focus:z-[100] focus:bg-white focus:px-4 focus:py-2 focus:type-body"
     >
       Skip to main content
     </a>
     <Header />
     <PageTransition>
       <main id="main-content">{children}</main>
     </PageTransition>
     <Footer />
   </body>
   ```

   **CRITICAL layout structure rules:**
   - Header and Footer are OUTSIDE PageTransition (they persist across routes, never fade)
   - PageTransition wraps ONLY `<main>` with `{children}` inside it
   - `<main>` has `id="main-content"` which the skip-to-content link targets
   - Skip-to-content link is the first element in `<body>` — `sr-only` by default, visible on focus
   - The skip link uses `z-[100]` to ensure it appears above everything when focused

   **Keep existing code unchanged:**
   - All font variable classNames on `<html>` stay as-is
   - Metadata export stays as-is
   - `import './globals.css'` stays as-is
