## Interaction Contracts

### Header Scroll Behavior (LAYO-03)

| State | Condition | Header Appearance |
|-------|-----------|-------------------|
| At top (hero) | scrollY < 80px | Transparent background. White-to-transparent gradient behind text only. |
| Scrolled (past hero) | scrollY >= 80px | Solid `#f0eeeb` (off-white) background. Full opacity. |
| Scrolling down | scrollY increasing and delta > 10px | Header slides up out of view. CSS `transform: translateY(-100%)` with `transition: transform 300ms ease`. |
| Scrolling up | scrollY decreasing and delta > 5px | Header slides back into view. `transform: translateY(0)`. |

Scroll threshold for show/hide: 80px (Claude's discretion — large enough to avoid jitter on small scrolls).
Show/hide delta threshold: 10px down, 5px up (prevents flicker on micro-scrolls).
Implementation: `'use client'` component with `useEffect` + `scroll` event listener. Pure CSS transitions only — no Framer Motion.

Source: CONTEXT.md D-04, D-05, D-06; Discretion applied for thresholds.

### Mobile Menu (LAYO-02)

| State | Trigger | Behavior |
|-------|---------|----------|
| Closed | Default | Hamburger icon visible (right-aligned in header). Menu hidden. |
| Opening | Hamburger click | Full-screen overlay fades in. CSS `opacity: 0 → 1`, `visibility: hidden → visible`, duration 300ms ease. |
| Open | After fade-in | Overlay white (#ffffff) fills viewport. "JENNIE SLADE" centered at top. Nav links stacked vertically centered, 48px gap between each. Instagram icon centered at bottom. X button top-right. |
| Closing | X click or nav link click | Overlay fades out (reverse). On link click: close first, then navigate (avoid flash). |

Hamburger icon: Three horizontal lines, 24px wide, 2px stroke, `#1a1a1a`. No animated X morph — plain swap to X icon on open.
Mobile breakpoint: below 768px (md breakpoint). Above 768px: desktop horizontal nav visible, hamburger hidden.

Source: CONTEXT.md D-07, D-08, D-09, D-10; Discretion applied for breakpoint.

### Page Transitions (DESN-06)

CSS opacity fade on route change. Apply to `<main>` content wrapper only — not the nav/footer (they persist).
Duration: 200ms. Easing: ease-in-out.
Implementation: Use Next.js App Router layout with a client wrapper component around `{children}` that applies a CSS animation class on mount.

Source: CONTEXT.md D-21.

### Button Hover States

| Button | Default | Hover |
|--------|---------|-------|
| Primary | Black bg (#000000), white text | 80% opacity (`opacity: 0.8`). Transition 150ms ease. |
| Secondary | Transparent bg, black border (1px solid #000000), black text | Black bg (#000000), white text. Transition 150ms ease. |

Source: CONTEXT.md D-17, D-18; CLAUDE.md button section.

## Accessibility Requirements

- Header `<nav>` element with `aria-label="Main navigation"`
- Mobile menu trigger: `aria-expanded="true/false"`, `aria-controls="mobile-menu"`
- Mobile menu overlay: `role="dialog"`, `aria-modal="true"`, `aria-label="Navigation menu"`
- Focus trap inside mobile menu when open (Tab cycles through nav links and X button only)
- `<footer>` semantic element
- All icon-only links have `aria-label` (hamburger, X, Instagram)
- Minimum touch target 44px for hamburger and X buttons (padding to compensate for visual icon size)
- Skip-to-content link at top of `<body>` linking to `#main-content` (add to root layout)

## What Claude's Discretion Resolved

| Decision Area | Value Chosen | Rationale |
|---------------|-------------|-----------|
| Mobile/desktop breakpoint | 768px (Tailwind `md:`) | Industry standard; Tailwind's `md` breakpoint aligns with typical phone/tablet boundary |
| Scroll threshold (hero → solid nav) | 80px | Generous enough for header height, avoids snapping too early |
| Scroll show/hide delta | 10px down, 5px up | Prevents jitter; asymmetric values reduce flicker on very small scroll reversals |
| Animation easing | ease / ease-in-out | Smooth, no bounce — consistent with editorial magazine feel |
| Footer copyright | Auto-generate with `new Date().getFullYear()` | Stays current without manual updates |
| Hamburger icon style | Lucide-react `Menu` + `X` icons | Zero-weight option; consistent with codebase if lucide is already present; otherwise inline SVG |
